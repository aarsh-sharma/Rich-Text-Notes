# Rich Text Notes

### Description

Add notes in a Rich Text Editor which are saved on a database.

### To Run

```cd``` into the directory and

```yarn install``` to install node_modules

```yarn start``` to run the application.

### Technologies

- React JS
- Redux
- Firebase
- ReactQuill (for editor)
